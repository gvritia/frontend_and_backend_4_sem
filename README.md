# Практическая работа №5: Расширенный REST API с Swagger

## Описание проекта
Практическая работа №5 находится в папке practice_4
Проект представляет собой демонстрационное веб-приложение с фронтендом на **React** и бэкендом на **Express**, реализующее CRUD для сущности **Пользователь (User)**.  
Документация API создаётся автоматически с помощью **Swagger (swagger-jsdoc + swagger-ui-express)**.

---

## Функционал

### Backend
- CRUD операции с пользователями:
  - **GET /api/users** — получить всех пользователей
  - **GET /api/users/:id** — получить пользователя по ID
  - **POST /api/users** — создать нового пользователя
  - **PATCH /api/users/:id** — обновить данные пользователя
  - **DELETE /api/users/:id** — удалить пользователя
- Swagger документация доступна по адресу: `http://localhost:3000/api-docs`
- Валидация обязательных полей: `name` и `age`
- Автоматическое логирование запросов (метод, путь, тело запроса)

### Frontend
- Страница **UsersPage**:
  - Отображение списка пользователей
  - Создание нового пользователя через модальное окно
  - Редактирование пользователя через модальное окно
  - Удаление пользователя
- Компоненты:
  - `UsersPage.jsx` — страница с логикой CRUD
  - `UserItem.jsx` — отображение одного пользователя
  - `UserModal.jsx` — модальное окно для создания/редактирования
- API интеграция через `axios` (файл `src/api/index.js`)
